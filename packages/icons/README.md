# `@viterian/icons`

> The icon components for react

## Install
```$ npm install @viterian/icons```
```$ yarn add @viterian/icons```

## Usage
```
import React from 'react';
import {Link} from '@viterian/icons';
function App() {
  return (
    <div className="App">
      <Link size={24} color={'blue'}></Link>
    </div>
  );
}

export default App;
```
